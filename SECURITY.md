# Security Policy

kindpath-collective-website is the public-facing website for KindPath Collective — built with Next.js and TypeScript.

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email: security@kindpathcollective.org

We will acknowledge receipt within 48 hours and coordinate responsible disclosure.

## Scope

Security concerns relevant to this repository include:
- Cross-site scripting (XSS) vulnerabilities in React components
- Exposed environment variables or API keys in client-side code
- Dependency vulnerabilities in npm packages
- Server-side rendering security issues
- Content Security Policy gaps
